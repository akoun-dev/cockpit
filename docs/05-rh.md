# Module Ressources Humaines

**Menu sidebar** : Ressources Humaines  
**Icône** : `Users`  
**Clé** : `rh`

## Rôle

Indicateurs RH : effectifs, absentéisme, formation, recrutement, performance, climat social.

## Composants utilisés

| Composant | Rôle |
|-----------|------|
| `ModuleHeroSection` | En-tête du module avec stats clés |
| `KpiModuleView` | Grille complète des indicateurs |
| `ModuleDocuments` | Documents liés au module |

## Sous-domaines typiques

- Effectifs
- Absentéisme
- Formation
- Recrutement
- Performance
- Climat social

## Contenu de la vue

1. **Hero section** : synthèse des indicateurs RH
2. **Cartes prioritaires** : KPIs RH clés
3. **Grille détaillée** : tous les indicateurs groupés par sous-domaine, chaque ligne avec nom, valeur, cible, écart, tendance, statut

## API

- `GET /api/indicators/domain?domain=RH&year=...&quarter=...` — indicateurs du module
